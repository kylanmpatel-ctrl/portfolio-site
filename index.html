This Portfolio

I built this entire website myself from scratch using HTML, CSS, and vanilla JavaScript. I had never really built a full website before so this was a big learning experience for me. I started by laying out the structure in HTML, then styled everything with CSS including the dark theme, animations, and making it look good on mobile. I used JavaScript for things like the scroll effects, the mobile menu, and the contact form. Once it was done I pushed everything to GitHub and deployed it on Vercel which makes it live on the internet. The hardest part was getting the layout to look clean on both desktop and mobile at the same time. I'm still adding to it as I learn more.
